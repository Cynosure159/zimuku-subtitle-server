# Zimuku Subtitle Server - 执行计划 (Execution Plan)

## 阶段 1：核心爬虫与 OCR 引擎开发 (关键路径)
**目标**: 实现无需 UI 的搜索与验证码绕过。
- [x] **Task 1.1**: 搭建 Python 虚拟环境，安装 `httpx`, `beautifulsoup4`, `lxml`。
- [x] **Task 1.2**: 实现 `SimpleOCR`。复刻像素采样逻辑，能够识别 Base64 验证码。
- [x] **Task 1.3**: 实现 `ZimukuAgent` 的核心搜索逻辑。支持多级页面解析。
- [x] **Task 1.4**: 实现验证码挑战机制。模拟重定向获取授权 Cookie。
- [x] **Task 1.5**: 编写测试脚本（`tests/test_scraper.py`），验证搜索与 OCR 识别率。

## 阶段 2：数据库与 FastAPI 基础架构
**目标**: 提供持久化支持与 REST 接口。
- [x] **Task 2.1**: 使用 `SQLModel` 定义数据模型（配置、缓存、任务）。
- [x] **Task 2.2**: 初始化 FastAPI 框架，配置全局异常处理与日志。
- [x] **Task 2.3**: 开发配置 API。支持在线修改代理地址与主域。
- [x] **Task 2.4**: 实现搜索 API（带 SQLite 缓存逻辑）。
- [x] **Task 2.5**: 编写测试脚本，验证数据库 CRUD。

## 阶段 3：下载服务与 MCP 协议集成
**目标**: 实现完整下载流程与 AI 代理支持。
- [x] **Task 3.1**: 开发下载路由。处理多步跳转下载。
- [x] **Task 3.2**: 开发 `ArchiveManager`。支持 ZIP/7z 解压与压缩包内乱码修正（`CP437` -> `GBK`）。
- [x] **Task 3.3**: 集成 MCP SDK。暴露搜索与下载为 MCP Tools。
- [x] **Task 3.4**: 手动在 Claude/Gemini 中通过 MCP 测试下载功能。

## 阶段 3.5：Web UI 增强型 API 开发
**目标**: 为前端面板提供完整的数据管理与系统监控能力。
- [x] **Task 3.5.1**: **任务管理 API**。实现 `GET /tasks`（列表/分页）、`DELETE /tasks/{id}`（删除记录）、`POST /tasks/{id}/retry`（重试失败任务）及 `POST /tasks/clear-completed`（一键清理）。
- [x] **Task 3.5.2**: **媒体库管理 API**。实现 `GET/POST/DELETE/PATCH /media/paths`，支持管理电影与剧集的扫描目录及其启用状态。
- [x] **Task 3.5.3**: **自动化匹配 API**。实现 `POST /media/match`，后端自动扫描目录并执行“视频-字幕”智能关联（当前为后台任务骨架）。
- [x] **Task 3.5.4**: **系统监控与统计 API**。实现 `GET /system/stats`（存储空间、任务统计）及 `GET /system/logs`（日志读取）。

## 阶段 4：Web UI 面板开发 (当前阶段)
**目标**: 构建现代化、易用的字幕管理后台。
- [ ] **Task 4.1**: **UI 视觉设计 (Pencil)**。
    - 绘制侧边栏导航、搜索页、电影/剧集管理页、任务页及设置页。
- [ ] **Task 4.2**: **初始化前端项目**。
    - 使用 Vite + React + Tailwind CSS + Lucide Icons 搭建基础架构。
- [ ] **Task 4.3**: **极简搜索页开发**。
    - 实现带语言筛选（简/繁/英/双语）的搜索功能。
    - 结果卡片展示：包含语言标签、来源、下载量及详情预览。
- [ ] **Task 4.4**: **媒体管理与匹配系统**。
    - **电影管理**: 支持配置扫描路径；提供“智能关联”视图，列出文件夹下的视频文件，一键搜索匹配并自动重命名（VideoName.srt）。
    - **剧集管理**: 支持配置剧集根目录；提供“季/集”筛选器，实现批量下载与重命名分发。
- [ ] **Task 4.5**: **任务管理与监控**。
    - 实时下载进度条、任务历史记录（重试/删除/清理）。
- [ ] **Task 4.6**: **系统全局配置**。
    - 代理设置、主域切换、OCR 引擎优先级、存储规则。

- [ ] **Task 5.3**: 完善 `README.md`。提供安装引导与 API 文档链接。

## 验收标准 (Definition of Done)
1. **成功搜索**: 输入“复仇者联盟”能返回带语言标识的字幕列表。
2. **自动下载**: 能够下载 ZIP 包并自动解压出 `.srt/.ass` 到 `/storage` 目录。
3. **乱码修复**: 解压后的中文文件名在现代 OS 上无乱码。
4. **验证码绕过**: 在触发验证码时，OCR 识别成功率不低于 85%，无需人工干预。
5. **API 文档**: Swagger UI (/docs) 自动生成。
