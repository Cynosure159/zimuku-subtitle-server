# Zimuku Subtitle Server (字幕库字幕服务)

一款独立的字幕管理与刮削服务，支持高效的 TV 剧集精确匹配、自动化媒体库扫描及 MCP（Model Context Protocol）协议集成，方便 AI 驱动实现自动化字幕管理。

## 核心特性

- **三层递进匹配策略**: 通过搜索页 → 季详情页 → 兜底模式，精准匹配剧集字幕
- **自动化媒体库扫描**: 自动检测电影和剧集文件，支持按是否有字幕进行筛选
- **MCP 协议集成**: 将搜索和下载功能暴露为 AI 可调用的工具，支持 AI 驱动自动化
- **多镜像自动切换**: 自动轮询所有可用下载镜像，确保下载可靠性
- **全面支持**: 支持电影和剧集
- **压缩包自动提取**: 支持 ZIP/7z 压缩包自动解压，智能识别编码 (CP437 → GBK)

## 技术栈

- **后端**: FastAPI + Python
- **数据库**: SQLite + SQLModel
- **前端**: React 19 + Vite + Tailwind CSS v4 + TypeScript

## 快速开始

### 后端

```bash
# 激活虚拟环境
source .venv/bin/activate

# 运行开发服务器
uvicorn app.main:app --reload
```

API 服务地址: `http://127.0.0.1:8000`
Swagger 文档: `http://127.0.0.1:8000/docs`

### 前端

```bash
cd frontend

# 安装依赖
npm install

# 运行开发服务器
npm run dev
```

前端服务地址: `http://localhost:5173`
